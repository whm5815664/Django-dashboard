# AIoT农业智能问答系统

## 系统概述

AIoT农业智能问答系统是一个基于Django框架开发的综合性农业知识服务平台，集成了先进的问答系统、知识库增强技术、图像识别和知识图谱抽取等核心功能。系统致力于为农业从业者提供精准、智能的农业知识问答服务，推动农业信息化和智能化发展。

## 核心功能模块

### 1. 智能问答系统

#### 基础问答功能
- **多模型支持**：集成DeepSeek-R1和Spark等多个大语言模型
- **上下文记忆**：支持多轮对话，保持对话连贯性
- **智能提示**：根据问题类型自动匹配专业提示模板
- **身份定制**：专门针对农业领域定制的AI助手身份

#### 知识库增强（RAG）技术
- **混合检索策略**：结合向量检索和BM25关键词检索，提高检索准确性
- **智能融合算法**：通过加权融合机制优化检索结果排序
- **质量控制**：设置多层质量门控，确保检索结果的可靠性
- **动态回退**：当知识库证据不足时，自动回退到纯LLM模式并明确提示

#### 知识库管理
- **结构化数据**：基于JSON格式的农业知识库，包含3万+条专业问答
- **实时更新**：支持知识库的动态增删改操作
- **可视化编辑**：提供友好的知识库管理界面

### 2. 图像识别系统

#### 病害识别
- **深度学习模型**：基于TC-MRSN预训练模型的高精度识别
- **多病害支持**：覆盖主要农作物病害类型
- **实时处理**：支持图片上传和即时识别
- **结果可视化**：提供直观的识别结果展示

### 3. 知识图谱抽取

#### 实体关系抽取
- **语义角色标注**：基于LTP工具进行深度语义分析
- **多策略抽取**：结合强触发和弱触发两种抽取策略
- **领域特化**：专门针对农业领域的作物-病害关系抽取
- **交互式可视化**：使用PyVis生成动态知识图谱

#### 智能推理
- **关系推断**：基于语义模式自动推断实体间关系
- **上下文理解**：结合句子上下文进行准确的关系识别
- **证据追溯**：为每个关系提供可追溯的证据链

### 4. 数据可视化大屏

#### 地理信息展示
- **多级地图**：支持全国、省级、基地级的多层次地理展示
- **数据集成**：整合柑橘产量、基地分布等农业数据
- **实时更新**：支持数据的动态加载和更新

#### 统计分析
- **产量分析**：提供柑橘等农作物的产量趋势分析
- **区域对比**：支持不同地区的农业数据对比
- **历史数据**：展示农业发展的历史轨迹

### 5. 辅助功能

#### 天气服务
- **实时天气**：集成天气API提供实时天气信息
- **区域监控**：支持多省份天气数据的批量监控
- **预警系统**：基于天气数据的农业预警功能

#### 视频监控
- **实时流媒体**：支持农业现场的实时视频监控
- **远程观察**：为远程农业管理提供可视化支持

## 技术创新点

### 1. 混合检索增强生成（Hybrid RAG）

**创新描述**：
- 融合向量检索和BM25关键词检索的双通道检索策略
- 通过α参数动态调节两种检索方式的权重
- 引入稳健归一化算法处理检索分数分布不均问题

**技术优势**：
- 提高检索准确性和召回率
- 适应不同类型的查询需求
- 降低单一检索策略的局限性

### 2. 智能质量控制机制

**创新描述**：
- 多维度质量门控：文档数量、最高分数、内容长度
- 词重叠度过滤：基于Jaccard相似度防止误匹配
- 通道配额管理：避免单一检索通道过度占用资源

**技术优势**：
- 确保RAG回答的可靠性
- 减少错误信息的传播
- 提升用户体验和信任度

### 3. 领域特化知识抽取

**创新描述**：
- 基于LTP的语义角色标注进行精确关系抽取
- 强触发和弱触发双重策略覆盖不同语义强度
- 作物-病害关系的专业化建模

**技术优势**：
- 提高农业领域实体关系抽取的准确性
- 支持复杂语义结构的关系识别
- 为农业知识图谱构建提供技术基础

### 4. 多模态智能融合

**创新描述**：
- 文本问答、图像识别、知识图谱的有机结合
- 跨模态信息的相互验证和补充
- 统一的农业知识服务平台

**技术优势**：
- 提供全方位的农业智能服务
- 增强系统的实用性和可靠性
- 满足不同场景的农业咨询需求

- **后端框架**：Django + Python

- **前端技术**：HTML5, CSS3, JavaScript, ECharts
- **数据存储**：JSON, 关系型数据库
- **部署方式**：支持本地和云端部署
